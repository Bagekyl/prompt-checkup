# Non-Prompt Test

## Input

```text
我最近想买一个机械键盘，预算 500 元左右，最好声音不要太吵，日常打字和写代码都能用，有没有什么推荐？
```

## Expected behavior

This should enter `non_prompt`. The workflow should explain that the input is a normal recommendation request rather than a prompt to diagnose, and ask the user to submit an actual prompt if they want a PromptCheckup report.
